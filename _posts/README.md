# Moonvault blog posts

Create posts in this directory with filenames in the form:

`YYYY-MM-DD-post-slug.md`

Start from `_drafts/post-template.md` and include at least:

```yaml
---
title: "Post title"
description: "A short summary used by the blog index and search engines."
---
```

Jekyll converts the Markdown into a themed page at:

`/blog/YYYY/MM/DD/post-slug/`
